# Get-YammerUsers

Query the Yammer API to get a list of (active, suspended, and deleted) Yammer users.

## Use case

Use this script to audit Yammer users.

## Run

Right-Click -> 'Run with PowerShell'

## Output

The script will automatically create a directory called 'Get-YammerUsers' in the users 'Documents' directory.  
In here, the script will create the report, log, and transcript file(s).
